# When to Use Numbered Lists in APA Style?

Numbered lists should be used when presenting items as complete sentences or paragraphs, such as: detailed conclusions, steps in a procedure, obtained results, or hypotheses. Numbered lists should be indicated with Arabic numerals followed by a period (e.g., 4.).

When to Use Bullet Point Lists in APA Style?
Bullet point lists are used to draw visual attention to items without implying a specific structure (e.g., when discussing topics like chronology, importance, or priority).

Use bullet points when the elements are phrases rather than complete sentences.

Not all bullet points are accepted in APA style. If you are using bullet points, the following symbols should be used:

Small circles
Squares
Dashes
If the bullet points are complete sentences, each point should begin with a capital letter and end with a period or other appropriate punctuation.

If the bullet points consist of words or phrases (but not complete sentences), each element should start with a lowercase letter (except for proper nouns or countries).

When to Use Alphabetical/Lettered Lists?
Use lettered lists to emphasize parallel elements within a sentence. Often, sentences contain lists of items that can be more clearly understood when separated by letters.

When a list within a sentence contains three or more elements, a comma should be used to separate them.

This type of list helps draw attention to elements and aids in understanding the distinct, parallel items.
