# Welcome to the Pre-Specialization Repository! 🎓

This document is intended for students enrolled in the pre-specialization 
program at Dr. Andrés Bello University. This repository is structured in a way that each module contains content that will provide real value to the learner. It is important to note that the program consists of five modules, two of which focus on writing

Please note that some of the documents are in Spanish and others in English. Apologies for any inconvenience, but I hope you find them useful and beneficial. Good luck and best wishes to all learners!"

### MODULE I: Technical Report Writing in Computer Science
This module contains information on:

🔗 Use of accent marks in capital letters.
🔗 What bullet points are and how to use them (Fundéu).
🔗 Enumeration (Wikilengua).
🔗 Spanish Language Dictionary (RAE).
🔗 Writing rules for prefixes (RAE).
Additionally, it covers the stages for professionally preparing a document.
